## Getting Started

### Dependencies

* Node.js >= 23.9.0
* npm >= 10.9.2

### Installing

1. Clone the repo
    ```sh
    git clone https://github.com/cloudiees/osu-score-dumper.git
    ```
2. Install NPM packages in both `osd-backend` and `osd-frontend`
    ```sh
    npm install
    ```

### Executing program

1. Run the backend in `osd-backend`
   ```sh
   npm run dev
   ```
2. Run the frontend in `osd-frontend`
   ```sh
   npm run dev
   ```
